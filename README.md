# Express-API w/ Cache

> Project developed as part of the [AppMasters][appmasters] selection test. It's an API that cache responses from [SuperHeroAPI][superheroapi].

- Demo. on Heroku: /search?q=bob

#### How-To 

- `/search?q=<search query>`
  - examples:
    - a
- `/hero/<slug-property>`
  - examples:
    - a
